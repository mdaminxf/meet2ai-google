## 🌟 Introduction

AI Classroom is a next-generation educational tool designed to simulate a real-world one-on-one tutoring session. Unlike standard chatbots, this application features a **virtual whiteboard** where the AI writes, draws diagrams, and solves problems step-by-step in real-time, synchronized with spoken explanations.

Whether you're learning physics, mathematics, coding, or history, the AI Teacher adapts its teaching style to provide visual and auditory learning experiences.

## 🧩 Problem Solving

Traditional AI chat interfaces have significant limitations for education:
1.  **Lack of Visuals:** Complex concepts (like circuits or geometry) are hard to explain with text alone.
2.  **Static Responses:** Answers appear instantly as a wall of text, which can be overwhelming.
3.  **No "Show Your Work":** Students miss the *process* of solving a problem.

**AI Classroom solves this by:**
*   **Dynamic Whiteboard:** Rendering diagrams and text in real-time.
*   **Voice Synchronization:** The AI speaks *while* it writes, mimicking a human teacher.
*   **Step-by-Step Execution:** Problems are broken down into logical steps (Setup, Strategy, Execution).
*   **Interactive Coding:** A dedicated code editor for programming tasks with line-by-line explanation.

| **Interaction** | Read & Reply | Interrupt, Screen Share, Voice Chat |
| **Pacing** | Instant output | Human-like pacing (writes as it speaks) |
| **Notes** | Copy-paste text | Download full PDF class notes |
